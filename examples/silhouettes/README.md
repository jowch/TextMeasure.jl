# Silhouettes

Procedural 2-D shape utilities for the TextMeasure.jl asteroid demo (`examples/asteroid_tui/`, #E).

| Function | Purpose |
|---|---|
| `asteroid_polygon(rng; n=12, lumpiness=0.4)` | Polar Perlin-noise asteroid silhouette (open CCW ring of `Point2{Float64}`). |
| `voronoi_shatter(polygon, impact; n_shards=4)` | Fracture a polygon into `n_shards ∈ [2,8]` shards that partition it. |
| `rasterize(polygon, cell_size)` | Rasterize to a `BitMatrix` (`row 1` = top, `col 1` = left; cell-center test). |

## Implementation notes

- `voronoi_shatter` builds the Voronoi tessellation with `DelaunayTriangulation.jl`
  (clipped to a convex bounding box so every cell is finite) and clips each **convex**
  cell to the parent via Sutherland–Hodgman. GeometryOps 0.1.40's polygon `intersection`
  is unreliable for this cell-clip (it drops overlapping cells and errors on
  edge-adjacency/containment), so the clip is done in-package; GeometryOps still backs
  orientation (`signed_area`) and the point-in-polygon test (`contains`).
- `rasterize`'s `row 1` = top / `col 1` = left convention matches `shape_pack`'s
  `raster_chord_fn` (#C) and `layout`'s block-top coordinate frame.

## Run the tests

```bash
julia --project=examples/silhouettes -e 'using Pkg; Pkg.instantiate(); Pkg.test()'
```
